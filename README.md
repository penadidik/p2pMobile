## About The Project

This project just example how to use couchbase to communication between 2 android apps without internet. One device as server and other as client.

This project have 1 main app and 3 module:
* The main app for implement three module
* First module is couchbase,this module to implement for [Couchbase](https://docs.couchbase.com/home/mobile.html)
* Second module is QR Code to implementation QR Code scanner
* Third module is utils to support development any app

Use the `BLANK_README.md` to get started.
